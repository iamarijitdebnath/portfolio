Welcome to the open-source repository of my personal portfolio. I am a software developer with a Master's degree in Computer Applications (MCA), passionate about backend architecture, API design, and creating scalable systems.

## ✨ Features

- **Modern Tech Stack**: Built using React 18, TypeScript, and Vite for lightning-fast performance.
- **Beautiful UI**: Styled with Tailwind CSS, utilizing a clean, minimal design aesthetic.
- **Dark/Light Mode**: Seamlessly switch between themes managed by a custom hook and CSS variables.
- **Fully Responsive**: Carefully optimized to look fantastic on desktop, tablet, and mobile screens.

## 🛠️ Tech Stack (Frontend)

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router DOM](https://reactrouter.com/)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have Node.js and npm installed on your machine.

### Installation

1. **Clone the repo**
   ```sh
   git clone https://github.com/iamarijitdebnath/portfolio.git
   ```

2. **Navigate to the directory**
   ```sh
   cd portfolio
   ```

3. **Install Packages**
   ```sh
   npm install
   ```

4. **Start the development server**
   ```sh
   npm run dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:5173` to see the application running!

## 📂 Project Structure

```text
src/
├── assets/         # Static assets and media
├── components/     # Reusable UI components (Header, Footer, Layout, ui/)
├── hooks/          # Custom React hooks (useTheme)
├── lib/            # Utility functions
├── pages/          # Individual route pages (Home, About, Projects, Contact)
├── App.tsx         # Main application container and routing
├── index.css       # Global styles and Tailwind configuration
└── main.tsx        # Application entry point
```
